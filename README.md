# JByteMod-Beta
JByteMod is a multifunctional bytecode editor. Including syntax highlighting and live decompiling.

JByteMod was originally based on JBytedit (Written in kotlin)

Libraries Used:
- Apache Commons IO
- Objectweb ASM
- RSyntaxTextArea
- SkidSuite
- Procyon Decompiler
- Fernflower Decompiler
